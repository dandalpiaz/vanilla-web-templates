Hi!

These vanilla-web-templates are a minimal, opinionated attempt to recreate common functionality needed in websites/apps using 'vanilla' CSS and JavaScript with an emphasis on native HTML elements, semantic structure, and accessibility.

Templates

  • Article / Blog Post – container-width
  • Contact Form – centered form
  • Login Form – centered form
  • Modal-based Editing – full-width, left-sidebar, modal, table
  • Grid – full-width, right-sidebar, grid
  • Posts Aggregator – full-width, right-sidebar, listing
